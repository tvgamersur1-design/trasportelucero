# Transporte Lucero - Website

Sitio web estático para Transporte Lucero, servicio de transporte y encomiendas entre Ferreñafe y Uyurpampa.

## 📁 Estructura del Proyecto

```
.
├── index.html          # Página principal (puedes renombrar lucero.html a index.html)
├── lucero.html         # Archivo principal actual
├── css/
│   └── styles.css      # Estilos y animaciones
├── js/
│   └── main.js         # JavaScript funcional (modal, animaciones)
├── img/
│   └── .gitkeep        # Carpeta para imágenes locales
└── README.md           # Documentación
```

## 🚀 Características

- ✨ Animaciones suaves al hacer scroll
- 📱 Diseño responsivo (móvil y desktop)
- 🎨 Material Design 3 con Tailwind CSS
- 📞 Integración directa con WhatsApp y llamadas telefónicas
- 🗺️ Google Maps integrado
- 🎯 Modal de selección de contacto

## 📝 Instalación

1. Clona o descarga el proyecto
2. Abre `lucero.html` en tu navegador
3. ¡Listo! No requiere instalación de dependencias

## 🎨 Personalización

### Cambiar colores
Edita las variables de color en el `<script id="tailwind-config">` en `lucero.html`

### Modificar animaciones
Edita los keyframes en `css/styles.css`

### Agregar funcionalidad
Agrega tus funciones en `js/main.js`

### Usar imágenes locales
1. Descarga las imágenes que necesites
2. Guárdalas en la carpeta `img/`
3. Actualiza las rutas de `src` en el HTML
